# zoommer-automation-tests

Automated UI tests for Zoommer website using Java, Selenide, TestNG, Page Object Model and Allure Report.

## Test coverage
- Product search test
- Category navigation test
- Product page and add to cart test

## Tech stack
- Java
- Maven
- Selenide
- TestNG
- Allure Report

## How to run tests
Use TestNG suite:
- testng.xml

## Reporting
Allure report can be generated from allure-results folder.
